# peristaltic_pump


## 项目简介
这是一个使用STM32控制的、由3D打印制作的蠕动泵项目。项目包括硬件设计和软件控制两部分。硬件部分包括一个3D打印的泵体和步进电机驱动电路；软件部分使用STM32的HAL库与FreeRTOS进行控制。现计划实现以下功能：
-  定速控制：通过设置目标速度，实现泵的定速运行。
-  定量控制：通过设置目标量，实现泵的定量泵送。
-  优雅的菜单：通过菜单界面，实现用户对泵的参数设置和运行控制。

未来规划：
- USB通信：实现通过USB串口与Python上位机进行通信，控制泵的运行。
